When would you want to use a remote repository rather than keeping all your work
local?

When you have more than one person working on the same files.
When you want to access the files from more than one computer.
To ensure the filed are backed up and always available.
When you want to make the files public.

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?


Forks: A clone within the github hosted environment.
Clones: A local copy of another repository. Thie could be another local repository or from github.
Branches: Another version of an existing repository. Branches can be merged etc.
